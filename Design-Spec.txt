Permanently Logged in to TASA Venmo Account
 - No need for Venmo authorization in the front end

Functionalities:
 - Set Start and End Date for searching transactions
 - Filter through entered keyword
 - Sum of transactions queried
 - Graphs of the total queried data

 Design:
 - TASA Bear Logo in the top-middle
 - Start and End Date text box with calendar drop-down option
 - Optional textbox to input filter keyword
 - Graphs will show up in the bottom after query
